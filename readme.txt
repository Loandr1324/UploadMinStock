1. Исходный файл должен называться "Анализ МО по компании.xlsx"
2. Исходный файл должен находится в этой же папки что и запускаемый файл "Загрузка МО.exe"
3. По результату работы создаётся файл "Загрузка МО по компании.xlsx"
4. Файл "Загрузка МО по компании.xlsx" можно загружать в любую базу на любое МО любого склада.
